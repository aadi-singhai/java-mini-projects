This is a simple Java console-based mini project that demonstrates:

Constructor usage
Static variables and methods
Auto-increment ID generation
Object creation and class-level data tracking
Each time a new Employee object is created:
A unique ID is automatically assigned.
The total number of employees increases.
Hiring is displayed in the console.

🚀 Features

Automatic employee ID generation
Static employee counter
Displays individual employee details
Displays total employee count using a static method
Clean console output formatting

🛠️ Technologies Used

Java (Core Java concepts)
OOP principles
Static keyword
Constructor overloading fundamentals
